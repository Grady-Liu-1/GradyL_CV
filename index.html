Research Assistant in Machine Learning applications for manufacturing

Ryerson University

Developed an application of Neural Network technology to be capable of using image classification. Using supervised learning to read 6 different graphs per training set that is paired with a value for flank wear. Trained with 140,000 artificially created training images from the original 112 training images based from the experimental data. The data consist of values collected from the sensors on a milling machine (vibration of the spindle, acoustics from the table spindle, the motor current, feedrate, depth of cut, etc) along with the type of material. This application was to assist the manufacturing industry by using machine learning to determine the condition of mill bits by the flank wear and when to change tools on milling machines. The network was coded using Python 2.7.15 and the graphs were done in MATLAB.
